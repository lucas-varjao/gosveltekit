# Frontend (SvelteKit)

Este frontend usa:

- SvelteKit (Svelte 5)
- Tailwind CSS
- shadcn-svelte (componentes UI)
- @lucide/svelte (ícones)
- Bun (runtime/tooling)

## Desenvolvimento

```bash
bun install
bun run dev
```

## Verificação

```bash
bun run check
```

## Base visual

- Tokens e atmosfera global em `src/app.css` (tema dark-only)
- Layout/padrões de página em `src/lib/components/layout/`
- Componentes de UI em `src/lib/components/ui/` (shadcn-svelte)
